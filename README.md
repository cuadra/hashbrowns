## Installation
Until the package is published on npm follow the steps below.

```javascript
//within the cloned repo
npm link
//within your projects folder
npm link hb
```

## File Usage
```javascript
hb package.json
//Renamed -> package.865e6603.json
```

## Folder Usage
```javascript
hb dist
//Renamed -> dist.4d1f4f4f
```

## Hash Only
```javascript
hb -h package.json
//865e6603
//returns hash without changing file
```
